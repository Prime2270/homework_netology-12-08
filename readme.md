```
2.1. Для резервирования данных в postgresql есть спец утилиты pg_dump, а так же pg_dumpall.
	Синтаксис pg_dump выглядит следующим образом: pg_dump <параметры> <имя базы> > <файл для сохранения копии>
	Так же для удобства и экономии дикового пространства можно сразу же сжимать дамп БД используя след синтаксис:
	pg_dump <параметры> <имя базы> | gzip > <файл для сохранения копии.gz>
	
	Для восстанавления можно воспользоватся стандартным синтаксисом psql <имя базы> < <имя копии базы.dump>
```
